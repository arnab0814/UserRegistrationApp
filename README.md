# User Registration App

A professional Android application for user registration with SQLite database integration.

## Features

✅ **User Registration Form** - Professional UI with modern design
✅ **Data Validation** - Email and input validation
✅ **SQLite Database** - Persistent data storage
✅ **Display Records** - View all registered users in table format
✅ **Navigation** - Previous/Next buttons to browse records
✅ **Clean UI** - Material Design with gradient colors

## Screenshots

### Registration Page
- Modern header with gradient background
- Styled input fields with icons
- Email validation
- Professional submit button

### Display Activity
- Table layout showing user data
- User counter (User 1 of 5, etc.)
- Navigation buttons
- Back to registration

## Technology Stack

- **Language:** Java
- **Database:** SQLite
- **Build System:** Gradle (Kotlin DSL)
- **Min SDK:** 23
- **Target SDK:** 36
- **UI Framework:** Android AppCompat
